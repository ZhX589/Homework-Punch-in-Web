# Homework-Punch-in-Web

一个功能完善的作业打卡平台，为教师和学生提供便捷的作业管理和提交功能。

## 项目简介

Homework-Punch-in-Web 是一个基于 Flask 框架开发的作业打卡平台，旨在简化教师和学生之间的作业管理流程。平台支持教师布置作业、上传附件，学生提交作业、上传图片等功能，并提供作业统计分析和提交情况查看等辅助功能。

## 功能特点

### 教师功能
- 布置作业，支持上传图片和文件附件
- 查看作业提交情况
- 对学生提交的作业进行评分
- 作业统计分析，包括提交率、评分分布等
- 添加和导入学生信息

### 学生功能
- 查看作业列表
- 提交作业，支持文本内容和图片上传
- 查看个人提交历史
- 组长可以查看同组同学的作业提交情况
- 组长可以对同组同学的作业进行评分

### 安全特性
- 用户认证和权限控制
- 文件上传安全检查
- 路径遍历攻击防护
- 唯一文件名生成，避免冲突

## 技术栈

- **后端框架**：Flask
- **数据库**：SQLite
- **前端**：HTML5, CSS3, JavaScript, Bootstrap
- **文件存储**：本地文件系统
- **安全**：密码加密，路径遍历防护

## 安装和运行

### 环境要求
- Python 3.7+
- pip 包管理工具

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/ZhX589/Homework-Punch-in-Web.git
   cd Homework-Punch-in-Web
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **运行应用**
   ```bash
   python run.py
   ```

4. **访问应用**
   打开浏览器，访问 `http://127.0.0.1:5000`

## 使用说明

### 登录
- **管理员**：用户名 `admin`，密码 `admin123`，班级 ID `000`
- **教师**：使用教师账号登录，密码为设置的密码
- **学生**：使用学生账号登录，密码为身份证后8位

### 教师操作
1. 登录后进入教师后台
2. 点击 "布置作业" 按钮创建新作业
3. 填写作业标题、内容、截止日期，可选择上传附件
4. 点击 "查看作业" 查看已布置的作业
5. 点击 "查看提交情况" 查看学生提交的作业
6. 对学生提交的作业进行评分

### 学生操作
1. 登录后进入学生后台
2. 查看作业列表，点击 "提交作业" 按钮提交作业
3. 填写提交内容，可选择上传图片
4. 点击 "我的提交" 查看个人提交历史
5. 组长可以点击 "查看同组作业" 查看同组同学的提交情况

## 安全配置

- **文件上传**：上传的文件存储在 `static/uploads` 目录中
- **文件安全**：系统会对上传的文件进行类型检查，只允许图片文件上传
- **路径安全**：系统实现了路径遍历防护，确保文件操作安全
- **密码安全**：教师密码使用 SHA-256 加密存储

## 常见问题

### 图片无法显示
- 检查图片格式是否为支持的格式（jpg, jpeg, png, gif）
- 检查服务器是否正常运行
- 检查文件上传目录权限是否正确

### 作业提交失败
- 检查是否已过截止日期
- 检查网络连接是否正常
- 检查上传的文件大小是否符合要求
